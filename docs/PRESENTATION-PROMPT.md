### Структура слайдов

#### Слайд 1 — Обложка
- Заголовок: **«Дворец горняков им. Ш. Дильдебаева»**
- Подзаголовок: «Цифровая платформа культурного центра · г. Сатпаев · 2026»
- Фоновая картинка: скрин hero-секции главной страницы
- Логотип/монограмма «Д» в золотом круге на navy-фоне

#### Слайд 2 — О заказчике
- КГКП «Центр культуры и творчества им. Ш. Дильдебаева» (бытовое — «Дворец горняков»)
- Адрес: 101300, г. Сатпаев, пр. К.И. Сатпаева, 106 (бывший к/т «Байконур»)
- История: 1974 → 2000 → 2019
- 22 творческих коллектива · 758 участников
- Народный ансамбль «Арман» с 1975 года
- Флагманские события: Республиканский конкурс кобызистов (с 2014), молодых поэтов (с 2017)

#### Слайд 3 — Проблема
- Нет единого информационного портала
- Заявки в кружки / на аренду залов — по телефону, теряются
- Нет онлайн-афиши, сложно узнать о событиях
- Двуязычный контент дублировать вручную
- Нет каналов соцсетей и автопостинга

#### Слайд 4 — Решение
- Современный двуязычный сайт (kk/ru) с адаптивом
- 7 публичных разделов: главная, новости, события, кружки, о нас, контакты, аренда залов
- 17 разделов админ-панели
- ИИ-чатбот с голосовым вводом (Gemini)
- Автопубликация в Telegram + Instagram
- Онлайн-записи, подписка на события, рекомендация кружков через ИИ

#### Слайд 5 — Технологический стек
Визуал: логотипы стека в сетке
- **Frontend:** Next.js 16 · React 19 · TypeScript 5 · Tailwind CSS 4
- **Backend:** Node.js 20 · PostgreSQL 16 (raw SQL через `pg`)
- **Auth:** JWT + Argon2 · RBAC (3 роли)
- **ИИ:** Google Gemini 2.0 Flash
- **Соцсети:** Telegram Bot API · Instagram Graph API v19
- **Инфра:** Docker Compose · Caddy (Let's Encrypt)

- HTTPS через Caddy + Let's Encrypt

- Адаптив для mobile/tablet/desktop
- 38 страниц, build зелёный, TS strict
