## Рекомендации по использованию

1. **Hero на главной** — поставить `dvorets-01.png` (фасад здания с памятником). Это узнаваемое изображение, оно отличает дворец от любого другого ДК и сразу создаёт доверие.
2. **Под `og:image`** — также `dvorets-01.png` (после ресайза до 1200×630 и обрезки).
3. **Карточки событий на главной** — раздать `dvorets-07/11/12` (сцены, концерты, дети).
4. **Кружки / студии** — `dvorets-11` (танцы), `dvorets-09-1` (КВН), `dvorets-04` (музыка), `dvorets-05` (вокал).
5. **«Наша история» / «О нас»** — `dvorets-01` + `dvorets-02` (директор).
6. **Перед заливкой в `/public`** — пережать в WebP/AVIF, добавить версии 400/800/1200w для `srcset`. Размеры исходников 300×201 — 613×409, для современного hero этого мало; для статичных карточек подойдут.
7. **Альтернативный текст** уже описан в таблице выше — переиспользуйте для `alt` атрибутов.
8. **Атрибуция** — в подвале указать «Фото: КГКП "Центр культуры и творчества им. Ш. Дильдебаева"» или связаться с Центром для официального разрешения.

## Юридическое замечание

Фотографии загружены с публичной страницы вышестоящей государственной организации (КГКП ОНМЦД), относятся к самому объекту съёмки (Дворец и его мероприятия). Для production-сайта рекомендуется:

1. Получить от Центра письменное согласие на использование архивных фото (одно письмо — закрывает вопрос).
2. Заказать профессиональную фотосессию здания (фасад / интерьер / залы) и мероприятий — это полностью снимает риски и даёт уникальный контент.
3. До получения собственных снимков использовать эти фото с подписью источника.
